How to connect to desktops

Spice client native apps

  • Windows 64bits: Install viewer - Install usb plug
  • Windows 32bits: Install viewer - Install usb plug
  • Debian/Ubuntu based distros: sudo apt install virt-viewer -y
  • RedHat/CentOS/Fedora based distros: sudo dnf install remote-viewer -y
  • Android: Install aSpice viewer
  • MacOS: There is no spice client app currently. Use default HTML5 browser option.

VPN network access & RDP (Wireguard VPN client to access guests IPs is required)

  • Install wireguard client . After installing wireguard client you'll need your config file to set up VPN.
  • Android: Not tested.
  • MacOS: Open downloaded console.vncloc file with default VNC client viewer app.
  • iOS: Not tested
